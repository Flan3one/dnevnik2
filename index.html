Помощь

Это приложение для ведения онлайн-журнала оценок.

Загрузка журнала - на этой вкладке пользователь может загрузить оценки из файла формата .csv, .txt, .xlsx.

Редактирование журнала - На этой вкладке пользователь может отредактировать загруженный ранее журнал.
Для удаления записи об ученике нужно нажать на кнопку "Удалить запись", а потом мышкой кликнуть на необходимую строку таблицы.
Для редактирования записи необходимо мышкой кликнуть на необходимую строку таблицы.

Статистика - На этих вкладках пользователь может просмотреть статистику по журналу в табличном и графическом виде.

О разработчике - На этой вкладке пользователь может найти информацию о создателе программы.
